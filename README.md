## Description
Order-service. Another project for personal playground for NestJS framework.

[![Build Status](https://travis-ci.org/anasceym/order-service.svg?branch=master)](https://travis-ci.org/anasceym/order-service)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
